# Quickstart for Create Class Periods

This document provides instructions on how to run the application and test the "Create Class Periods" feature.

## Running the Application

1.  **Install dependencies**:
    ```bash
    pnpm install
    ```

2.  **Run the development server**:
    ```bash
    pnpm dev
    ```

3.  **Access the application**:
    Open your web browser and navigate to the URL provided by the development server (usually `http://localhost:5173`).

## Testing the Feature

1.  Navigate to the "Classes" section of the application.
2.  Select a class to view its schedule.
3.  Click on the "Add Period" button in the `ScheduleView`.
4.  The "Add Period" dialog should appear.
5.  Follow the on-screen instructions to create a single or repeating class period.
